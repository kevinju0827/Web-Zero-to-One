# M04 List and Table

## Goals

Organize structured data effectively using ordered/unordered lists and tables.

## CheckPoints

* [ ] Use AI to generate a data table (e.g., a schedule or feature comparison) and manually check for `<table>`, `<tr>`, `<th>`, and `<td>` tags.
* [ ] Create an unordered list (`<ul>` and `<li>`) for a set of items.
* [ ] Create an ordered list (`<ol>` and `<li>`) for a step-by-step process.
