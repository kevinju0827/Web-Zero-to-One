# M11 Position

## Goals

Break out of the normal document flow to place elements exactly where they are needed.

## CheckPoints

* [ ] Apply position: relative to a parent container element.
* [ ] Apply position: absolute to a child element to place it precisely within the relative parent.
* [ ] Create a `sticky` or `fixed` element (like a top navbar or a "back to top" button) that stays in place while scrolling.
