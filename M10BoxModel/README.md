# M10 Box Model

## Goals

Master the core of CSS layout by understanding margins, borders, padding, and content areas.

## CheckPoints

* [ ] Use Google Chrome Developer Tools to inspect the Box Model of at least three different elements.
* [ ] Apply margin to create appropriate spacing between your `<div>` containers.
* [ ] Apply padding to create breathing room inside your `<div>` containers.
* [ ] Add a temporary border to your elements to visually verify the Box Model boundaries.
