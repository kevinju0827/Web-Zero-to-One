# M15 Bootstrap

## Goals

Speed up development and ensure consistency by utilizing the world's most popular frontend framework.

## CheckPoints

* [ ] Include Bootstrap 5 in your project via a CDN link in the `<head>` section.
* [ ] Implement Bootstrap's grid system classes (.container, .row, .col) to build a responsive layout quickly.
* [ ] Replace custom-built UI components with at least two pre-built Bootstrap components (e.g., a Navbar and Buttons).
