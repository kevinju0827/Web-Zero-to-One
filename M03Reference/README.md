# M03 Reference

## Goals

Connect pages and embed visual media to make the website interactive and engaging.

## CheckPoints

* [ ] Insert at least one `<a>` tag that links to an external website, using the `target="_blank"` attribute.
* [ ] Embed at least one image using the `<img>` tag with a visual asset sourced from Pixabay or unDraw.
* [ ] Verify that every `<img>` tag includes a descriptive alt attribute for accessibility.
