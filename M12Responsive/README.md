# M12 Responsive

## Goals

Ensure the website looks great and functions well on all devices, from mobile phones to desktop monitors.

## CheckPoints

* [ ] Write at least one CSS @media query to change the layout for screens smaller than 768 px (mobile devices).
* [ ] Test the responsiveness of your site using the device toolbar in Chrome's Developer Tools.
* [ ] Ensure text remains readable and images do not overflow their containers on mobile viewports.
