# M02 Common Elements

## Goals

Learn semantic HTML by applying headings and paragraphs to structure your project's content logically.

## CheckPoints

* [ ] Ensure the document contains exactly one `<h1>` tag representing the main topic.
* [ ] Use `<h2>` and `<h3>` tags to create a clear, logical structural hierarchy for your subtopics.
* [ ] Include at least three `<p>` tags containing descriptions or AI-generated article.
